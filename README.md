Quick start

1. Create a virtual environment (recommended):

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies (if any):

```bash
pip install -r requirements.txt
```

3. Run the CLI:

```bash
python -m src.main
# show version
python -m src.main --version
```
